## 二、7 天執行排程

## Day 1｜技術盤點 + 索引基線
- [ ] 到 Google Search Console（GSC）確認網域屬性已驗證。
- [ ] 提交 Sitemap：`https://btgamevip.com/sitemap.xml`
- [ ] 檢查 robots：`https://btgamevip.com/robots.txt`
- [ ] 記錄基線：
  - 已索引頁數
  - 近 28 天曝光 / 點擊 / CTR / 平均排名
  - 主要查詢詞（前 20）

## Day 2｜內容地圖與 URL 規劃
- [ ] 建立內容地圖（交易詞 × 資訊詞）。
- [ ] 規劃 3 篇文章 URL（已提供草稿）：
  - `.../0-1-zhe-shou-you-shou-chu-li-bao-ling-qu-zhi-nan/`
  - `.../shou-you-chong-zhi-sheng-qian-ce-lue-2026/`
  - `.../xin-shou-ru-keng-bt-shou-you-ping-tai-xuan-ze/`
- [ ] 每篇鎖定 1 主詞 + 3~5 次詞。

## Day 3｜上線第 1 篇（交易型）
- [ ] 發布「0.1折手遊首儲禮包領取指南」。
- [ ] 首段 120 字內出現主關鍵詞。
- [ ] 補 2~3 個站內連結 + 1 個首頁連結。
- [ ] 用 GSC URL Inspection 請求索引。

## Day 4｜上線第 2 篇（攻略型）

- [ ] 加入 checklist 與 FAQ 區塊。
- [ ] 導內鏈到第 1 篇與舊文。
- [ ] 索引請求 + 記錄收錄狀態。

## Day 5｜上線第 3 篇（新手教育型）
- [ ] 發布「新手入坑 BT 手遊平台選擇指南」。
- [ ] 加入「避坑清單」提高停留時間。
- [ ] 補上 related links 區塊，形成內容群。

## Day 6｜內鏈強化 + CTR 優化
- [ ] 舊文章補 2~3 條指向新文內鏈。
- [ ] 優化標題（含數字/年份/效益詞）。
- [ ] 優化 meta description（80~140 字）。
- [ ] 檢查首頁是否露出最新內容入口。

## Day 7｜索引複盤 + 下週待辦
- [ ] GSC 檢查：
  - 3 篇新文是否已索引
  - 哪些查詢詞開始有 impression
- [ ] 若未索引：排查 canonical/noindex/重複內容/薄內容。
- [ ] 產出下週主題（至少 5 篇）與關鍵詞補強清單。

## 三、內容主題規則（寫作模板）

每篇固定結構：
1. 問題切入（痛點）
2. 步驟式解法（3~6 點）
3. 常見誤區
4. FAQ（2~4 題）
5. CTA（導向首頁/相關文章）

字數建議：900~1400 字（短期先求「完整可解題」）。

---

## 四、內鏈規則（Internal Linking SOP）

1. 每篇新文至少 3 條內鏈：
   - 1 條 → 首頁
   - 1 條 → 同群主題文章
   - 1 條 → 既有舊文
2. 錨文字要語義化，避免都用「點此」。
3. 同一篇內不要重複同 URL 超過 2 次。
4. 優先鏈到：
   - 可轉換頁（首頁 / 優惠頁）
   - 同意圖內容（攻略↔攻略、優惠↔優惠）
5. 發新文後 48 小時內，回頭補舊文反向連結。

---

## 五、索引檢查步驟（Indexing Checklist）

1. **可抓取性**
   - robots.txt 未封鎖文章路徑
   - 頁面回應碼 200
2. **可索引性**
   - 頁面無 `noindex`
   - canonical 指向自身 URL
3. **可發現性**
   - 已出現在 sitemap.xml
   - 至少有 1 條站內連結可到達
4. **提交與追蹤**
   - GSC URL Inspection → Request indexing
   - 24~72 小時後再次查驗
5. **未收錄排查**
   - 內容過短/重複
   - 標題與內容關聯弱
   - 內鏈太少導致權重不足

---

## 六、KPI（7 天短週期）
- 新增索引頁：+3
- 新增可見查詢詞：+20（含長尾）
- 平均 CTR 提升：+0.3%~1%
- 每篇文章至少帶來 impression（即使點擊仍低）

> 備註：短期先追「收錄與曝光」，第 2~4 週再追「排名與點擊」。
