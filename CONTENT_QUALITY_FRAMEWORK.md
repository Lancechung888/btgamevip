# 內容品質框架（btgamevip）

## 一篇文只解一個問題
- 範例：
  - ✅ 新手首儲怎麼不浪費（3步）
  - ❌ 手遊全攻略大全

## 開頭三句公式
1. 痛點：你現在最常犯的錯
2. 後果：繼續這樣會損失什麼
3. 承諾：看完這篇能得到什麼

## 固定結構模板
1. H1（明確結果）
2. H2 常見錯誤（2~4點）
3. H2 可執行步驟（3~5步）
4. H2 快速檢查清單（Checklist）
5. H2 FAQ（3題）
6. 結尾 CTA（單一動作：前往 btgamevip.com）

## 文字規範
- 每段 2~4 行
- 每段至少有 1 個動作句（先做 / 不要做 / 今天就做）
- 先給結論，再給理由

## SEO 規範
- 主關鍵字：標題、首段、1個H2、結尾
- 次關鍵字：自然出現，避免硬塞
- 內鏈：每篇 2~3 個（首頁 + 2篇相關文）

## 發佈前 7 點檢查
- [ ] 標題有利益點
- [ ] 首段有痛點與承諾
- [ ] 有 3~5 個可執行步驟
- [ ] 有錯誤示例 vs 正確做法
- [ ] 有 FAQ（3題）
- [ ] 有內鏈
- [ ] CTA 單一明確
